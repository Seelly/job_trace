# Job Trace - 求职投递记录同步插件

 
一款帮助求职者自动抓取职位信息并同步到飞书多维表格的 Chrome 浏览器插件

 

---

## 📖 项目简介

在求职过程中，我们经常需要在多个招聘平台浏览职位并投递简历。手动整理投递记录不仅繁琐，还容易遗漏重要信息。**Job Trace** 通过浏览器插件的方式，帮助你：

- 🎯 **一键抓取**：在任意职位详情页自动提取公司、岗位、地点等关键信息
- ✏️ **灵活编辑**：通过悬浮窗实时修正和补充抓取结果
- 📊 **统一管理**：将所有投递记录同步到飞书多维表格，建立统一的求职追踪台账
- 🔄 **自动同步**：配置一次，后续投递自动记录

## ✨ 功能特性

### 核心功能

- **智能抓取**：自动识别页面中的职位信息（公司、岗位、地点、链接等）
- **悬浮窗编辑**：在页面右侧显示悬浮窗，支持实时修正字段
- **飞书集成**：直接写入飞书多维表格，无需手动复制粘贴
- **字段映射**：支持自定义插件字段与飞书表格字段的映射关系
- **置信度提示**：显示抓取置信度，帮助判断是否需要人工确认

### 支持的数据源

插件通过以下方式提取职位信息：

- 页面标题和元信息
- 结构化数据（JSON-LD）
- 页面标题标签（h1/h2/h3）
- 关键词匹配

适用于大多数招聘网站和公司官网的职位页面。

## 🚀 快速开始

### 前置要求

- Chrome 浏览器（版本 88+）
- 飞书账号
- 飞书开放平台应用（需要多维表格权限）

### 安装步骤

#### 1. 下载插件

```bash
git clone https://github.com/yourusername/job-trace-chrome-plugin.git
cd job-trace-chrome-plugin
```

#### 2. 加载到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录
6. 在浏览器工具栏中固定该插件

#### 3. 准备飞书多维表格

**方式一：使用模板（推荐）**

复制飞书多维表格模板：[点击这里](https://my.feishu.cn/base/QAAeb2cllaZezVswQozcGmVPnfg?table=tblI8T5rhuporKBn&view=vewxvWV222)

**方式二：自建表格**

创建包含以下字段的多维表格：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 公司 | 文本 | 公司名称 |
| 岗位 | 文本 | 职位名称 |
| 投递方式 | 单选 | 投递渠道 |
| BASE | 多选 | 工作地点 |
| 投递链接 | 超链接 | 职位链接 |
| 当前状态 | 单选 | 投递状态 |
| 投递时间 | 日期 | 投递时间 |
| jd | 文本 | 职位描述 |

#### 4. 创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 开通"多维表格"权限
4. 获取 `App ID` 和 `App Secret`

#### 5. 配置插件

1. 点击插件图标，在悬浮窗右上角点击"设置"
2. 填写飞书应用信息：
   - App ID
   - App Secret
   - appToken（从飞书多维表格链接中获取）
   - tableId（从飞书多维表格链接中获取）
3. 配置字段映射（如果使用模板则无需修改）
4. 点击"测试连接"验证配置

## 📝 使用指南

### 基本流程

1. **打开职位页面**：访问任意招聘网站的职位详情页
2. **显示悬浮窗**：点击浏览器工具栏中的插件图标
3. **抓取信息**：点击悬浮窗中的"抓取当前页面"按钮
4. **确认修正**：检查抓取结果，必要时手动修正字段
5. **写入飞书**：点击"写入飞书"按钮完成同步

### 字段映射配置

默认字段映射关系：

```
插件字段 -> 飞书字段
company -> 公司
title -> 岗位
deliveryMethod -> 投递方式
location -> BASE
jobUrl -> 投递链接
status -> 当前状态
appliedDate -> 投递时间
notes -> jd
```

如果你的飞书表格字段名与默认不同，可以在设置页自定义映射。

### 常见问题

<details>
<summary><b>抓取结果不准确怎么办？</b></summary>

不同网站的页面结构差异较大，插件可能无法完美提取所有字段。你可以：
- 在悬浮窗中手动修正字段
- 等待页面完全加载后重试
- 对于特定网站，可以提交 Issue 请求增强支持
</details>

<details>
<summary><b>写入飞书失败？</b></summary>

请检查：
- 飞书应用是否有权限访问目标表格
- appToken 和 tableId 是否正确
- 字段映射是否与飞书表结构一致
- 网络连接是否正常
</details>

<details>
<summary><b>悬浮窗没有显示？</b></summary>

- 确保已点击插件图标激活
- 刷新页面后重试
- 检查浏览器控制台是否有错误信息
</details>
